# 使用方法


- 将此库克隆到assloud-app根目录，形成如下结构


|- assloud-app
    |- assloud-installer

- 将Assloud编译后，拷贝到application目录，形成如下结构

|- assloud-app
    |- assloud-installer
        |- _package
            |- application
                |- Assloud.exe
                |- ...
        |- ...

- 运行make.bat
- 在_output目录得到安装文件Assloud-Installer.exe
